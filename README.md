# Zpaste
-
### A command line tool   that list all used paste string on Mac OS .



## Use:

- download the Binary file https://github.com/yokee99/Zpaste/releases
- mv it to the ~/usr/local/bin/

```
$sudo mv ./Zpaste ~/usr/local/bin/
```

- run :

```
$Zpaste -d 
```

- list :

```
$Zpaste 
```


### Enjoy!
